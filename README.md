# Shop Online - Frontend

Ứng dụng thương mại điện tử (E-Commerce) xây dựng bằng React.js.

## Công nghệ sử dụng

- **React 19** - Framework UI
- **Redux** - Quản lý state
- **Ant Design 5** - UI Component Library
- **React Router DOM v7** - Routing
- **Axios** - HTTP Client

## Cài đặt & Chạy local

```bash
npm install --legacy-peer-deps
npm start
```

Mở [http://localhost:3000](http://localhost:3000) để xem ứng dụng.

## Biến môi trường

Tạo file `.env` tại thư mục gốc:

```env
REACT_APP_API_URL=https://soleup-backend-production.up.railway.app
```

## Build production

```bash
npm run build
```
